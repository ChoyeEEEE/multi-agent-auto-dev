# 🤖 Multi-Agent Auto Dev System

A multi-agent AI system that automatically generates, tests, and debugs code.

## 🚀 Features
- Planner Agent: task decomposition
- Coder Agent: code generation
- Tester Agent: execution & validation
- Debugger Agent: automatic bug fixing

## 🔁 Workflow
1. User input task
2. Planner breaks it down
3. Coder generates code
4. Tester runs code
5. Debugger fixes errors (loop)

## 🧠 Architecture
Multi-agent collaboration with iterative self-improvement loop.

## 🛠 Tech Stack
- Python
- OpenAI API
